ApiClient

This class is the main communication class for the API library.

Please refer to constructor and public methods for usage.

package

Topcheck\ApiClient

Constructor, builds the object.

__construct(string $login, string $password, array|null $testServer = null)

Arguments

$login

string

The login for TopCheck API server.

$password

string

The password for TopCheck API server.

$testServer

array|null

Optional. Array of key-value pairs with test server info. Array should contains at least one key and two optional keys. 'host' - Full address (including protocol) of the test server to be used (required). Production one will be used if none provided. 'htlogin' - Login for basic auth (optional). 'htpass' - Pass for basic auth (optional).
